Instructions

The data must be prepared in a certain way before it is uploaded to Statsomat: Each row must represent a record respectively an observation. Each column must represent a variable respectively a parameter of interest. Do not store multiple variables in one column (for example sex and age in one column). Leave cells blank in case of missing values. Other values for missing values (e.g. NAs) are not recognized as such. Please take a look at these prepared dataset examples, they can help you with the data preparation.

Once you have your data prepared in the specified way, please save it with the text CSV format. If your data file is in an Excel or LibreOffice format, you can try to convert it to CSV by selecting "Save As". Try to select the format CSV with the character encoding UFT-8, this helps reading the data correctly. You can also prepare or inspect the CSV file with a text editor like Notepad++.

When your data is ready, you can proceed with the upload to Statsomat. A couple of upload properties must be specified. If you are not sure about them, you can choose the option "Auto" for each property. In that case, Statsomat will try to identify automatically the right settings. In most of the cases, this will succeed. Otherwise, please consider alternative settings or data formats.

After a successful upload of the data, you can proceed with the selection of the variables which should be considered by this app.

Now you can generate and download the report. To make use of the table of contents, please open the report with a PDF reader (for example Adobe Acrobat Reader DC).
